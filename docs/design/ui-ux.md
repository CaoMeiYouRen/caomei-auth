# UI/UX 设计规范

本文档定义了草梅 Auth 的视觉识别系统 (VI) 和用户体验准则。

## 1. 视觉风格

项目采用现代、明快且具有安全感的视觉风格，基调以红色为主，象征活力与警示。

### 主题色规范

| 变量名            | 色值      | 用法                   |
| :---------------- | :-------- | :--------------------- |
| **Primary**       | `#e63946` | 主按钮、链接、激活状态 |
| **Primary Light** | `#ff6b6b` | 悬停状态、浅色背景元素 |
| **Primary Dark**  | `#a52834` | 点击状态、深色背景元素 |
| **Background**    | `#f8fafc` | 默认页面背景           |
| **Surface**       | `#ffffff` | 卡片、输入框背景       |

### 调色板 (PrimeVue Palette)

-   50: #ffe5e9
-   500: #e63946 (Base)
-   950: #3a0c12

## 2. 图标与字体

-   **图标库**: 统一使用 `@mdi/font` (Material Design Icons)。
-   **字体**: 优先使用系统默认无衬线字体堆栈。

## 3. 布局与响应式

-   **断点**: 遵循 Tailwind/PrimeFlex 标准。
-   **移动端优先**: 保证所有功能在移动端浏览器下均有完整的操作体验。
-   **管理后台**: 采用侧边栏/顶栏混合布局，支持折叠。

## 4. 组件规范

-   **表单**:
    -   必须包含校验状态反馈（错误提示、成功状态）。
    -   关键操作需有 Loading 状态拦截重复点击。
-   **反馈**:
    -   成功操作使用绿色 Toast。
    -   错误异常使用红色 Banner 或对话框。
