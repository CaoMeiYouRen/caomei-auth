# 项目路线图 (Roadmap)

本文档概述了草梅 Auth 的中长期发展方向和大型功能规划。

## 阶段 1: 稳定性与核心增强 (当前阶段)

_目标：打造稳健、安全的统一认证基石。_

-   [x] OAuth2.0 协议全量支持。
-   [x] 基础 MFA (多因子认证) 支持。
-   [ ] **多语言支持 (i18n)**: 全球化第一步。
-   [ ] **完善的自动化测试体系**: 保证代码质量。

## 2. 阶段 2: 开发者生态与集成优化

_目标：提升集成便利性，让开发者更快接入。_

-   [ ] **SDK 开发**: 提供 JS/TS, Python, Go 等主流语言的轻量级集成 SDK。
-   [ ] **插件系统**: 支持开发者编写自定义逻辑插件（如：自定义登录校验）。
-   [ ] **更多社交登录**: 集成钉钉、飞书、企业微信等国内办公场景登录。

## 阶段 3: 安全增强与智能化管理

_目标：通过 AI 和高级安全审计提升平台安全性。_

-   [ ] **异常登录检测 (AI)**: 基于地理位置、设备指纹的风险分析。
-   [ ] **异地登录告警**: 邮件/短信实时通知用户。
-   [ ] **自定义审计日志**: 支持将日志导出至外部安全分析系统 (SIEM)。

## 阶段 4: 企业级特性

_目标：面向大型组织和云原生环境。_

-   [ ] **组织机构管理**: 支持多级组织、部门结构。
-   [ ] **RBAC 深度集成**: 自定义角色与细粒度权限模型。
-   [ ] **多租户支持 (SaaS)**: 允许在一个实例下管理多个独立的租户环境。
